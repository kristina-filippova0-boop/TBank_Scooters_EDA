# Анализ сервиса аренды самокатов Т-Банка

## О проекте
Проект представляет собой **разведочный анализ данных (EDA)** сервиса аренды самокатов Т-Банка.  
Цель — выявить поведенческие паттерны пользователей, проверить продуктовые гипотезы и подготовить визуализации для презентации результатов.

Использован датасет с поездками (~400 тыс. наблюдений), содержащий:
- Время и длительность поездок
- Стоимость и начисленный кэшбэк
- Возраст и семейное положение клиентов
- Регионы и модели самокатов

---

## Цели анализа
- Провести EDA: распределения, сезонность, выбросы
- Построить визуализации для ключевых инсайтов
- Сформулировать продуктовые гипотезы
- Оценить долю сервиса Т-Банка на рынке

---

## Основные шаги

1. **Загрузка и очистка данных**
   - Преобразование дат
   - Расчет длительности поездок
   - Удаление аномалий (поездки >50 км, длительность >12 часов)

2. **Разведочный анализ данных**
   - Активность пользователей по часам и месяцам
   - Анализ семейного положения и регионов
   - Распределение кэшбэка, выручки и повторных пользователей
   - Расчет корреляций между дистанцией, длительностью и выручкой

3. **Визуализации**
   - Сезонность поездок
   - Среднее количество поездок по семейному положению
   - Топ-10 регионов по количеству поездок
   - Средний чек по дням недели
   - Распределение выручки и корреляции

## Продуктовые гипотезы
1. **Кэшбек увеличивает активность**  
   Пользователи с кэшбеком делают больше поездок по сравнению с теми, кто его не получает:contentReference[oaicite:0]{index=0}.

2. **Модель самоката влияет на стоимость поездки**  
   Средние и медианные значения по моделям различаются от 25,7 руб. (Lucky) до 110,5 руб. (ES200):contentReference[oaicite:1]{index=1}.

3. **Активность пользователей по часам**  
   Пиковая активность — часы 5, 14–16, а не 8–10 и 17–19, как предполагалось ранее:contentReference[oaicite:2]{index=2}.

4. **Длительность поездки коррелирует с дистанцией**  
   Корреляция log(distance_km) и log(ride_duration_min) = 0,79, что подтверждает прямую зависимость:contentReference[oaicite:3]{index=3}.

5. **Новые пользователи делают короткие поездки**  
   Постоянные клиенты совершают более длинные поездки, чем новые:contentReference[oaicite:4]{index=4}.

6. **Выручка на пользователя коррелирует с количеством поездок**  
   Сильная положительная корреляция (0,976): больше поездок → больше дохода:contentReference[oaicite:5]{index=5}.
